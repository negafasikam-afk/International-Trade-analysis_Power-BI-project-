# Canada's International Trade Report

This repository contains the Power BI dashboard and PySpark code for the **Updated Trade Dashboard project**, which tracks Canada’s investment and trade data.

## Overview
The project includes:  
- **Power BI Dashboard**: Visualizes trends, balances, and quarterly performance of Canadian investment and trade data.  
- **PySpark ETL Scripts**: Implements a **Medallion architecture** in a Lakehouse environment to clean, transform, and organize data for reporting.  

Data is sourced from **Statistics Canada**. The ETL process ensures reliable, high-quality data for analytics and reporting.

## Repository Structure
- `PowerBI_Report/` – Contains the Power BI file and screenshots.  
- `PySpark_Code/` – PySpark scripts for data cleaning, transformation, and pipeline processes.  
- `Data/` – Information about datasets used (no sensitive data included).  

## Technologies Used
- **Power BI**: Interactive dashboards and data visualization  
- **Python / PySpark**: Data cleaning, transformation, ETL  
- **Microsoft Fabric / Lakehouse**: Data storage and Medallion architecture  
- **SQL**: Data querying and management
